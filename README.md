# simreq.py

- This library is similar to requests, but in a single file version.

## Example tests

```py
import simreq
c = simreq.get('https://api.jiroawesome.tech')
print(c.status_code)
```

### Note
- You must download or copy the file (simreq.py) first before importing it.
